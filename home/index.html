Sistema de gestión y medición de la huella de carbonoUPM2022Se desarrolló un sistema de gestión y medición de la huella de carbono utilizando placas ESP32, equipadas con medidores de corriente para monitorear el consumo eléctrico de los dispositivos. El sistema cuenta con sensores de movimiento que detectan cuando un aula está vacía; si se detecta consumo eléctrico durante la ausencia de personas, se apagan automáticamente todos los dispositivos conectados para evitar el consumo innecesario. Además, el sistema está integrado con Telegram para enviar notificaciones sobre las mediciones de consumo y permitir el apagado remoto de dispositivos a través de mensajes. Esta solución contribuye a reducir la huella de carbono mediante la optimización del consumo energético en espacios educativos.

Para ello el sistema incluye sensores de velocidad, infrarrojos y giroscopios, entre otros, para monitorear y evaluar el comportamiento del conductor. Esta solución está diseñada para mejorar la seguridad al volante, reduciendo el riesgo de accidentes causados por distracciones.
